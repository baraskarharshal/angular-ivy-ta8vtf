Key Points

  • You can define custom events that send observable output data from a child to a parent component.
  • The HTTP module uses observables to handle AJAX requests and responses.
  • The Router and Forms modules use observables to listen for and respond to user-input events.
  • Event Emitter extends Rxjs Subject observable.
  • It can be used with @Output decorator to emit value from component.
  • Event Emitter can be defined in a service and then it can be subscribed in multiple component.
